## Project abstract
问题：基于豆瓣推荐的分层挖掘频繁模式/关联规则

目的：利用豆瓣关于某个电影推荐的十个电影，发现在不同抽象层上的强关联规则的分布/密度，从而利用不同抽象层上的强关联规则探索豆瓣自己的推荐更多的依赖于什么（可行性？？？在实践中再确定）

实现路径：原始数据获取 -> 数据预处理 -> 挖掘频繁模式/关联规则 -> 数据可视化
